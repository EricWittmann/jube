General Questions

What is the license?

Jube uses the Apache 2.0 License.

What is it?

Jube is a pure Java open source emulator of Kubernetes for use on non-Linux platforms.

If you are using Linux we highly recommend you use Kubernetes! However if you must run Java middleware on non-Linux platforms then Jube can help provide a kubernetes-like experience on platforms where golang, Kubernetes or Docker are not supported,

Jube lets you use fabric8 on any platform that supports a JVM.

What does jube do?

Jube orchestrates creating Java processes from Image Zips.

What Java versions is supported?

Jube runs on Java 7 and 8.

Which platforms are supported?

Any platform that has a Java Virtual Machine
